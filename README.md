# 智能政务助手

> 数智赋能政务 · 便民服务零距离

一个基于 **HTML5 + Layui + ECharts + 讯飞星火大模型 + 高德地图** 构建的智能政务服务平台前端项目，涵盖政务服务全流程：从智能问答、在线预约、办事申报到数据分析与数据开放，为市民提供一站式"互联网+政务"体验。

![首页](docs/images/home-top.png)

---

## 项目简介

本项目以"让群众少跑腿、数据多跑路"为核心理念，打造了一个功能完备的智能政务助手系统。前端采用纯静态架构（无需后端服务），通过 LocalStorage 模拟数据库、讯飞星火大模型提供 AI 能力、高德地图 API 提供地理信息服务，实现了政务服务的数字化、智能化。

系统包含 **20+ 独立功能页面**，覆盖政务服务的方方面面，同时融入课程思政元素，强调职业素养与家国情怀。

## 技术栈

| 类别 | 技术 | 说明 |
|------|------|------|
| **UI 框架** | Layui | 提供 Table、Form、Layer、Dropdown 等企业级组件 |
| **数据可视化** | ECharts | 折线图、柱状图、饼图、雷达图等图表 |
| **AI 能力** | 讯飞星火大模型 (WebSocket) | 智能问答、语义理解、知识库匹配 |
| **地图服务** | 高德地图 JS API 2.0 | 实时地图展示、网点标注、2D/3D 切换 |
| **加密** | CryptoJS (AES) | 大模型通信数据加密 |
| **数据存储** | LocalStorage | 模拟关系型数据库，实现 Session 级状态保持 |
| **天气 API** | 高德天气接口 | 首页天气展示 |
| **前端架构** | 原生 HTML/CSS/JavaScript | 语义化标签、CSS 变量体系、模块化组件 |

## 功能模块

### 1. 用户认证系统

提供完整的登录/注册流程，支持手机号+密码登录、记住密码等功能。登录页面采用左右分栏设计，左侧为品牌展示区，右侧为登录表单区。

![登录页](docs/images/login.png)

### 2. 智能首页

首页采用沉浸式 Hero 设计，集成以下功能入口：

- **实时天气展示**：调用高德天气 API 显示当前天气
- **智能搜索**：支持关键词搜索政务事项
- **六大功能入口**：办件进度、预约取号、智能问答、政策文件、办事指南、反馈建议
- **办结通知**：实时推送办件审核结果
- **人生一件事·套餐办**：出生、入学、置业、退休四大场景一键联办
- **数据与特色服务**：便民服务地图、政策计算器、政务数据分析、政府数据开放

### 3. "一件事"套餐办

围绕市民生命周期关键节点，提供四个"一件事"联办场景：

| 场景 | 覆盖服务 |
|------|----------|
| 出生一件事 | 出生医学证明、预防接种证、户口登记、医保参保 |
| 入学一件事 | 学区查询、学位申请、网上报名、录取结果查询 |
| 置业一件事 | 购房资格审核、房产交易网签、公积金贷款、不动产登记 |
| 退休一件事 | 养老金申领核验、公积金提取、老年优待证办理 |

每个场景提供可视化的进度追踪面板，展示各子事项的完成状态，支持一键联办申请。

## 快速开始

### 环境要求

- 现代浏览器（Chrome 80+ / Edge 80+ / Firefox 78+）
- 无需 Node.js 或任何构建工具
- 无需后端服务器

### 启动方式

1. **克隆项目**

```bash
git clone https://github.com/your-username/智能政务助手.git
cd 智能政务助手
```

2. **直接打开**

由于项目为纯静态前端，有多种启动方式：

- 直接双击 `login.html` 在浏览器中打开
- 使用 VS Code 的 Live Server 插件
- 使用任意 HTTP 服务器（如 Python）：

```bash
# Python 3
python -m http.server 8080

# 然后访问 http://localhost:8080/login.html
```

> **注意**：AI 问答功能需要有效的讯飞星火 API 凭证。如需使用，请在 `ai.js` 中配置您的 APPID 和 API Secret。

## 设计特色

- **政务蓝配色体系**：采用 `#1890ff` 为主色调，搭配 `#001529` 深蓝背景，体现政务严谨专业的视觉风格
- **CSS 变量系统**：全局统一的 Design Token，支持多主题切换（深蓝政务 / 简约浅灰 / 党建红）
- **Glassmorphism 导航栏**：毛玻璃效果导航，提升视觉层次感
- **沉浸式 Hero 区域**：渐变背景 + 动态粒子，营造科技感
- **响应式布局**：适配不同屏幕尺寸
- **无障碍支持**：语音交互模式，适合老年人使用
